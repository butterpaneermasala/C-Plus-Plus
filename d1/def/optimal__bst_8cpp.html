Detailed Description

Implementation of [Optimal Binary Search Tree (OBST)] (https://www.geeksforgeeks.org/dynamic-programming-set-24-optimal-binary-search-tree/)

Given a sorted array of keys and their corresponding frequencies, the task is to construct a binary search tree with minimum cost. The cost of a BST node is defined as frequency multiplied by depth.

Algorithm

This uses dynamic programming with prefix sum optimization to reduce redundant computation. It builds up the solution for increasing subtree sizes. Time complexity is O(n^3) due to three nested loops.
